Question-1: What are the different ways to select an element in the DOM?

Ans:

There are 5-6 ways to select an element in Dom.

  • GetElementByID
  • GetElementsByClass
  • GetElementsByTagName
  • querySelector(class/id)
  • querySelectorAll(class)
  • GetElementByName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:

innerHTML:
This selects the html code inside the selected element.
innerText:
This selects the text inside the selected element.
textContent:
This selects the text along with the html code inside the selected element.
